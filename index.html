Welcome to ​StadiaIcons Shortcuts​!

What is it?

​StadiaIcons​ is a set of game icons designed with ​Google Stadia​ in mind. Each icon follows a simple design principle inspired by the Stadia logo and colors.

As for ​StadiaIcons Shortcuts​, it allows you to easily add individual Stadia game shortcuts with their corresponding ​StadiaIcon​ to your devices in just a few clicks.

How does it work?

It's really quite simple: select a Stadia game you want to install from the list below and we'll guide you through the process.
